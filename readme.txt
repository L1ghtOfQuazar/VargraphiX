#VargraphiX
	Приложение для операционной системы Windows, предназначенное для работы с таблицами данных. 
	Таблица формата csv загружается в приложение, после чего с ней можно работать, используя различные методы графического анализа данных.
#Установка приложения
	1) Сохраните все файлы в одну папку.
	2) Установите Python версии 3.9.7 из приложенного в GitHub файла.
	   Примечание: для работы программы необходимо, чтобы указанная версия python была установленна корректно, 
	   а также была единственной в системе.
	3) Установите Octave GUI из приложенного в GitHub файла.
	   Примечание: Запустите консоль CMD и введите команду "octave". В случае запуска GUI программа установленна корректно.
		       Вторым методом проверки правильности установки octave является запуск файла testgui.py. 
		       (CMD должна быть открыта в папке, где лежат все файлы)
                       Запустите его двойным нажатием левой кнопки мыши. В случае запуска GUI программа установленна корректно.
	4) Запустите файл moduledwnld.py двойным нажатием левой кнопки мыши. 
	   Примечание: Вы также можете установить необходимые модули вручную, прописав в CMD "pip install requirements.txt" 
	   (CMD должна быть открыта в папке, где лежат все файлы)
	5) В случае корректной установки, запустите файл main.pyw или ярлык VargraphiX двойным нажатием левой кнопки мыши.
#Тестовые файлы
	Для освоения программы VargraphiX пользователем мы предоставляем табличный файл testtable.csv, лежащий в папке вместе с проектом.
	ВАЖНО! При работе с программой следует использовать csv файлы, оформленные по следующему стандарту: Разделитель - ',', Кодировка - 'utf-8'.
